# Siemens MindSphere GraphQL API

Siemens MindSphere is an industrial IoT platform. The API covers asset management, time series data ingestion and retrieval, IoT connectivity, analytics pipelines, digital twin models, event management, file service, and edge device management for manufacturing and infrastructure.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Siemens MindSphere REST API
**Documentation:** https://documentation.mindsphere.io/MindSphere/

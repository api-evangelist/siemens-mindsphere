### Siemens MindSphere Industrial IoT API
Siemens MindSphere provides industrial IoT platform APIs for machine connectivity, time-series data storage, asset management, and edge analytics. REST and OData APIs enable telemetry ingestion, fleet monitoring, predictive maintenance, and digital twin modeling for manufacturing equipment.

The MindSphere Asset Management API enables creating and managing assets, defining aspects and asset types, configuring data models, and handling connectivity for IoT devices. Supports hierarchical asset structures for digital twin modeling.

### Siemens MindSphere IoT Time Series API
The MindSphere IoT Time Series API enables storing, querying, and retrieving time-series telemetry data from industrial equipment. Supports high-frequency data ingestion, bulk imports, and time-range queries for sensor and machine data.

### Siemens MindSphere MindConnect API
The MindSphere MindConnect API provides device connectivity for ingesting telemetry data from industrial equipment into the MindSphere platform. Supports REST and MQTT protocols for hardware agents including MindConnect Nano, IoT2040, IoT2050, and software-based agents.

**Human URL:** [https://documentation.mindsphere.io/MindSphere/apis/connectivity-mindconnectapi/api-mindconnectapi-overview.html](https://documentation.mindsphere.io/MindSphere/apis/connectivity-mindconnectapi/api-mindconnectapi-overview.html)

The MindSphere Integrated Data Lake API provides storage and retrieval of large-scale industrial data. Supports OData contracts for standardized data access, cross-account data sharing, and bulk data transformation operations.
